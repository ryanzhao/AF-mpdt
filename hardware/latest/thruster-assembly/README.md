# Thruster Assembly

## Function

The thruster assembly subsystem contains the MPD thruster body and related mechanical parts, including electrodes, nozzle geometry, clamps, plates, separators, and magnet holder components.

## Included CAD files

- `mpd-v1.step`: Main MPD thruster assembly STEP export.
- `mpd-thruster draft 1.stl`: Mesh export of the MPD thruster draft for 3D-printing or mesh-based review.
- `Anode nozzle 2.step`: Anode nozzle STEP export.
- `rocket nozzle (1).step`: Rocket nozzle STEP export.
- `anode plate.step`: Anode plate STEP export.
- `anode conducting plate.step`: Anode conducting plate STEP export.
- `hollow cathode.step`: Hollow cathode STEP export.
- `hollow cathode (1).step`: Alternate or companion hollow cathode STEP export from the latest uploaded set.
- `cathode clamp.step`: Cathode clamp STEP export.
- `w70cu clamp.step`: W70Cu clamp STEP export.
- `right plate.step`: Right plate STEP export.
- `right plate (1).step`: Alternate or companion right plate STEP export from the latest uploaded set.
- `seperator.step`: Separator STEP export, retaining the uploaded filename spelling.
- `Magnet holder (2).step`: Magnet holder STEP export.
- `Magnet holder (2).3mf`: Magnet holder 3D manufacturing file for slicing or print-related workflows.

## File format notes

- `.f3d`: Original Autodesk Fusion 360 design file. No `.f3d` source file is included in this uploaded set.
- `.step`: Neutral CAD exchange file for use across CAD platforms.
- `.stl`: Mesh export commonly used for 3D printing.
- `.3mf`: 3D manufacturing file that may include slicing, material, or print-related metadata.
