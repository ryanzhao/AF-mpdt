# AF-MPD Thruster 项目描述

## 项目动机

AF-MPD（Applied-Field Magnetoplasmadynamic）thruster 是一种利用外加磁场增强等离子体加速能力的电推进概念。项目目标是完成一个可制造、可测试、可迭代的 AF-MPD 推进器原型，并围绕该原型建立真空测试、结构支撑、供电控制与数据采集的基础平台。通过小型化样机验证，可以为后续推力、放电稳定性、热管理与结构可靠性优化提供工程依据。

## AF-MPD thruster 工作原理简介

AF-MPD thruster 通过阴极与阳极之间的电弧放电将推进剂电离为等离子体。放电电流与磁场相互作用产生洛伦兹力，使等离子体沿喷管方向加速排出，从而形成推力。与自场 MPD 推进器相比，AF-MPD 额外引入外加磁场，以增强等离子体约束和加速效果，并改善低功率或小尺寸样机中的放电稳定性。

## 系统组成

- **Thruster**：包含阴极、阳极喷口、绝缘/分隔结构、夹具以及导电板等部件，是放电与等离子体加速的核心组件。
- **Vacuum chamber**：提供低压测试环境，降低空气放电与大气阻力对实验结果的影响，并尽量模拟空间电推进工作条件。
- **Test platform**：用于固定推进器、布置供气/供电/测量接口，并支持后续推力测量、热监测和放电观测。
- **Control panel**：集中管理供电、触发、推进剂流量、仪表读数和安全联锁，便于测试流程重复执行。
- **Frame**：为真空舱外部设备、控制面板和测试平台提供机械支撑，并便于运输、装配和维护。

## 设计迭代过程

当前仓库保留了多个 CAD 版本与零部件模型，反映了推进器从初始草图到分体部件设计的迭代过程。早期设计侧重于验证喷管、阴极和阳极几何关系；后续版本进一步拆分出阳极导电板、夹具、磁体支架、分隔件和喷管部件，以便分别加工、装配和替换。后续迭代应继续围绕装配公差、绝缘距离、热膨胀、磁场布置和维修便利性进行优化。

## 测试方法

1. 在真空舱中安装 thruster 与 test platform，并确认机械固定、绝缘间隙和电气连接。
2. 对 vacuum chamber 抽真空，记录基准压力，并检查泄漏与放电安全条件。
3. 通过 control panel 设置供电、推进剂流量、触发流程和数据记录参数。
4. 进行低功率点火测试，逐步提高电流、电压或推进剂流量，观察放电稳定性、弧根位置、热响应和可见等离子体形态。
5. 记录电压、电流、压力、流量、温度、图像/视频以及任何推力测量数据，并在每轮测试后检查电极烧蚀和结构损伤。

## 当前结果

当前阶段已经形成推进器核心部件与支撑部件的 CAD 资产，并完成了项目资料目录整理。仓库中的现有 CAD/设计文件已集中放入 `images/cad-renders/`，便于后续添加渲染图、装配图与制造图。测试平台照片、项目总览图、海报图和系统结构图的目录已经预留，后续可按用途继续补充。

## 限制和未来改进方向

- **实验数据不足**：需要补充真空测试记录、放电曲线、推力估算、温度数据和图像证据。
- **渲染与照片资料不足**：需要导出 CAD 渲染图，并补充测试平台照片、项目总览图、海报图和系统结构图。
- **热管理风险**：电极和夹具可能在高电流测试中快速升温，需要完善材料选择、散热路径和测试占空比。
- **绝缘与安全联锁**：高压/大电流测试应进一步明确绝缘距离、接地策略、急停和互锁逻辑。
- **磁场优化**：需要结合仿真或实测磁场分布，优化磁体支架、磁场强度和磁场方向。
- **可测量性改进**：未来应加入标准化传感器布置、推力架或摆架、同步数据采集脚本和测试报告模板。
